23. SMA andalan mempunyai 735 siswa. Jumlah siswa kelas X dan XI adalah 525 orang, sedangkan jumlah siswa kelas XI dan XII Adalah 455 orang hitunglah banyak siswa tiap kelas. ​.


Jawaban:

Banyak siswa tiap kelas

Kelas X = 280 siswa

Kelas XI = 245 siswa

Kelas XII = 210 siswa

Penjelasan dengan langkah-langkah:

Misalkan:

Kelas X = a

Kelas XI = b

Kelas XII = c

Jumlah siswa SMA = 735 siswa

a + b + c = 735 siswa (persamaan 1)

Jumlah X dan XI = 525 siswa

a + b = 525 siswa (persamaan 2(

Jumlah XI dan XII = 455 siswa

b + c = 455 siswa (persamaan 3)

Substitusi persamaan 2 ke persamaan 1

a + b + c = 735

525 + c = 735

c = 210 (persamaan 4)

Substitusi persamaan 4 ke persamaan 3

b + c = 455

b + 210 = 455

b = 245 (persamaan 5)

Substitusi persamaan 5 ke persamaan 2

a + b = 525

a + 245 = 525

a = 280

Banyak siswa tiap kelas

a = Kelas X = 280 siswa

b = Kelas XI = 245 siswa

c = Kelas XII = 210 siswa

Harap diperhatikan substitusi tidak perlu sama persis seperti begini, jika mau kamu bisa subtitusi persamaan 3 ke 2 dan substitusi lainnya juga, yang penting itu salah satu variabelnya hilang.

31. Umur rata rata gabungan siswa kelas X dan kelas XI adalah 16,5 tahun. Jika umur rata rata siswa kelas X adalah 16 tahun dan umur rata rata siswa lelas XI adalah 17 tahun, maka perbandingan banyaknya siswa kelas X dan banyaknya siswa kelas XI adalah


Jawaban:

1 : 1

Penjelasan:

misal

[tex]x_{gab}[/tex] = rata-rata gabungan = 16,5

[tex]x_1[/tex] = rata-rata umur kelas X = 16

[tex]x_2[/tex] = rata-rata umur kelas XI = 17

[tex]n_1[/tex] = banyak siswa kelas X

[tex]n_2[/tex] = banyak siswa kelas XI

dengan rumus rata-rata gabungan didapat

[tex]x_{gab}=\frac{x_1n_1+x_2n_2}{n_1+n_2}\\\\16,5=\frac{16n_1+17n_2}{n_1+n_2}\\\\16,5n_1+16,5n_2=16n_1+17n_2}\\\\16,5n_1-16n_1=17n_2-16,5n_2\\\\0,5n_1=0,5n_2\\\\\frac{n_1}{n_2}=\frac{0,5}{0,5}=\frac{1}{1}[/tex]

jadi perbandingannya adalah 1 : 1
