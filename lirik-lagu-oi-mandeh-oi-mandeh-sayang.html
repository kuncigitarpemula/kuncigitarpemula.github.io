Himpunan =

A - B = Selisih Himpunan

A n B = Irisan Himpunan

A u B = Gabungan Himpunan

Soal :

P = {2,3,4,5,6}

Q = {3,5,7}

R = {4,5,6,7}

Tentukan

a. (P - Q) U R

b. (P - Q) n R

Penyelesaian :

P = {2,3,4,5,6}

Q = {3,5,7}

R = {4,5,6,7}

(P - Q) U R

(2,4,6) U R

=> (P - Q) U R = {2,4,5,6,7}
